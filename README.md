# meow

## 项目简介
铲屎官助手，具有以下功能：
1. 记录宠物出生日、体重等基本信息
2. 添加驱虫、洗澡等提醒事项，可以设置提醒频率
3. 记录主粮、罐头这些生活用品的库存情况，显示有效期限、过期情况等

## 技术概述
使用vue全家桶，vue-cli、vue-router、vuex，结合cube-UI组件库。
